# 串口通信协议

DLC 固件与上位机（ASCOM/INDI 驱动）通过串口通信。

## 帧格式

所有命令和响应由 `<` 和 `>` 包裹：

```
发送: <命令>
响应: <响应内容>
```

## 完整命令列表

### 盖子控制

| 命令 | 参数 | 响应 | 说明 |
|------|------|------|------|
| `O` | 无 | 回显 `O` | 开盖 |
| `C` | 无 | 回显 `C` | 关盖 |
| `H` | 无 | 回显 `H` | 停止移动 |
| `P` | 无 | `0`–`5` | 查询盖子状态 |

**盖子状态码：**
| 值 | 含义 |
|----|------|
| 0 | NotPresent（未安装） |
| 1 | Closed（已关闭） |
| 2 | Moving（移动中） |
| 3 | Open（已打开） |
| 4 | Unknown（未知） |
| 5 | Error（错误） |

### 平场板控制

| 命令 | 参数 | 响应 | 说明 |
|------|------|------|------|
| `L` | 无 | `0`–`5` | 查询平场板状态 |
| `B` | 无 | 数值 | 查询当前亮度 |
| `M` | 无 | 数值 | 查询最大亮度 |
| `T<N>` | 亮度值 | 回显 `T<N>` | 开灯并设亮度 |
| `F` | 无 | 回显 `F` | 关灯 |
| `A` | 无 | 回显 `A` | 设置 autoON=true |
| `a` | 无 | 回显 `a` | 设置 autoON=false |
| `S<N>` | 毫秒 | 回显 `S<N>` | 设置稳定时间 |

**平场板状态码：**
| 值 | 含义 |
|----|------|
| 0 | NotPresent |
| 1 | Off |
| 2 | NotReady |
| 3 | Ready |
| 4 | Unknown |
| 5 | Error |

### 宽带/窄带预设

| 命令 | 参数 | 响应 | 说明 |
|------|------|------|------|
| `DB` | 无 | 回显 `DB` | 保存当前亮度为宽带值 |
| `DN` | 无 | 回显 `DN` | 保存当前亮度为窄带值 |
| `GB` | 无 | 亮度值 | 跳转到宽带值 |
| `GN` | 无 | 亮度值 | 跳转到窄带值 |

### 除露加热控制

| 命令 | 参数 | 响应 | 说明 |
|------|------|------|------|
| `R` | 无 | `0`–`6` | 查询加热器状态 |
| `W` | 无 | 回显 `W` | 手动开启（带限时） |
| `w` | 无 | 回显 `w` | 关闭所有加热 |
| `Q` | 无 | 回显 `Q` | 设置 autoHeat=true |
| `q` | 无 | 回显 `q` | 设置 autoHeat=false |
| `E` | 无 | 回显 `E` | 设置 heatOnClose=true |
| `e` | 无 | 回显 `e` | 设置 heatOnClose=false |
| `Y` | 无 | 传感器数据 | 查询温湿度详情 |

**加热器状态码：**
| 值 | 含义 |
|----|------|
| 0 | NotPresent |
| 1 | Off |
| 2 | Auto |
| 3 | On |
| 4 | Unknown |
| 5 | Error |
| 6 | Set (HeatOnClose) |

### 🆕 舵机实时控制

| 命令 | 参数 | 响应 | 说明 |
|------|------|------|------|
| `UO<N>` | 角度 (0–180) | 回显 | 设置主舵机开盖角度（写入 EEPROM） |
| `UC<N>` | 角度 (0–180) | 回显 | 设置主舵机关盖角度（写入 EEPROM） |
| `VO<N>` | 角度 (0–180) | 回显 | 设置副舵机开盖角度 |
| `VC<N>` | 角度 (0–180) | 回显 | 设置副舵机关盖角度 |
| `uO` | 无 | 角度值 | 查询主舵机开盖角度 |
| `i` | 无 | 角度值 | 查询主舵机关盖角度 |
| `vO` | 无 | 角度值 | 查询副舵机开盖角度 |
| `vC` | 无 | 角度值 | 查询副舵机关盖角度 |
| `J<N>` | 角度 (0–180) | 回显 | 🆕 **Jog 主舵机**至指定角度（直驱，不改变保存值） |
| `j` | 无 | 角度值 | 🆕 查询主舵机当前物理位置 |
| `K<N>` | 角度 (0–180) | 回显 | 🆕 **Jog 副舵机**至指定角度 |
| `k` | 无 | 角度值 | 🆕 查询副舵机当前物理位置 |

> ⚠️ `VO`/`VC`/`vO`/`vC` 仅在固件编译了 `SECONDARY_SERVO_INSTALLED` 时有效，否则返回 `?`。

### 系统命令

| 命令 | 参数 | 响应 | 说明 |
|------|------|------|------|
| `V` | 无 | 版本号 | 查询固件版本 |
| `Z` | 无 | `?` | 握手测试 |

## 通信流程示例

```
PC → 设备: <P>
设备 → PC: <1>           (盖子状态 = Closed)

PC → 设备: <O>           (开盖)
设备 → PC: <O>           (确认)

PC → 设备: <P>
设备 → PC: <2>           (状态 = Moving)

... 等待 ...

PC → 设备: <P>
设备 → PC: <3>           (状态 = Open)

PC → 设备: <UO45>        (设开盖角度为 45°)
设备 → PC: <UO45>        (确认，已写入 EEPROM)
```
